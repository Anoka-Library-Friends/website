## events.html

| Line | Type | What to replace |
|------|------|-----------------|
| 53 | SETUP | Google Calendar embed URL — replace `REPLACE_WITH_CALENDAR_ID` with the calendar's ID from Google Calendar Settings → Integrate calendar → Calendar ID |
| 68 | CONTENT | Footer address |
| 85 | CONTENT | Social media link — replace `#` |

**How to get the Calendar ID:**
1. Open Google Calendar → Settings (gear icon) → Settings
2. Click the calendar name under "Settings for my calendars"
3. Scroll to "Integrate calendar" → copy the Calendar ID (ends in `@group.calendar.google.com`)
4. The full embed src becomes: `https://calendar.google.com/calendar/embed?src=YOUR_ID%40group.calendar.google.com&ctz=America%2FChicago`
